## 什么是 Agent

- Agent = LLM + 目标（任务）+ 工具（调用外部能力）+ 状态/记忆（上下文、持久化存储等）
- 典型闭环：观察（输入）→ 计划/决策 → 行动（工具调用）→ 结果反馈 → 继续迭代直到完成任务
