## 什么是 Agent

Agent 可以理解为能在特定环境中完成任务的 LLM 系统：在目标驱动下，进行计划与执行，通过工
具调用与外部世界交互，并维护状态/记忆（短期上下文、长期存储等）来迭代完成任务。
